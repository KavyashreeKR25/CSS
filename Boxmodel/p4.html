CSS BOX MODEL

Every element that can be displayed on a web pages is comprised of one or more rectangular boxes. css box model typically describes how these rectangular boxes are laid out on a wed page. These boxes can have different properties and can interact with each other in different ways,but every box has a content area and optional surrounding padding border , and margin areas.
